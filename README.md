# Emergent Hardware Verification — Companion Code

The reference actor framework and every worked example from the book
**[Emergent Hardware Verification](https://emergenthardware.com)** by Bala Veluchamy.

- `actor_pkg/` — the SystemVerilog actor framework (the base substrate)
- `actor_pkg_cpp/`, `actor_pkg_systemc/` — the C++ and SystemC renderings
- `ch*_*/`, `app*_*/` — per-chapter and per-appendix examples (see `CODE_MAP.md`)

Browse all of it online, folder by folder, at **[emergenthardware.com/code](https://emergenthardware.com/code)**.
